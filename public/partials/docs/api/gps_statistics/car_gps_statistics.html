GET
/v1/cars/:car_id/gps_statistics
This will return gps_statistics if any associated with the car matching the id :car_id
GET
/v1/cars/:car_id/gps_statistics/:id
This will the gps_statistic matching with the id given
POST
/v1/cars/:car_id/gps_statistics
This create a new gps_statistic for the car matching :car_id
And a reply with the new gps_statistic object
