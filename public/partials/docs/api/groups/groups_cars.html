GET
/v1/groups/:group_id/cars
This will return the cars in the group matching :group_id
GET
/v1/groups/:group_id/cars/:id
This will return the car in the group matching :group_id with the matching id
POST
/v1/groups/:group_id/cars
This create a new car for the group matching :group_id Here is the POST data
And a reply with the new car object
PUT
/v1/groups/:group_id/cars/:id
This will update the car with the matching :group_id and :id
And a reply with the updated car
DELETE
/v1/groups/:group_id/cars/:id
This will delete the car with the matching :group_id and :id and respond with the deleted car object
