GET
/v1/groups/:group_id/users
This will return all the users in the group that matches :group_id
GET
/v1/groups/:group_id/users/:id
This will the user in the group matching :group_id matching :id
PUT
/v1/groups/:group_id/users/:id
This update the user matching :id
And a reply with the updated user object
DELETE
/v1/groups/:group_id/:id
This will delete the user with matching :id and reply with the deleted user object
