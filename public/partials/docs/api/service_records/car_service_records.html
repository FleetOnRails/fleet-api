GET
/v1/cars/:car_id/service_records
This will return service_records if any associated with the car matching :car_id
GET
/v1/cars/:car_id/service_records/:id
This will the service_record matching with the id given
POST
/v1/cars/:car_id/service_records
This create a new service_record for the car matching :car_id
And a reply with the new fuel_entry object
PUT
/v1/cars/:car_id/service_records/:id
This update the service_record for the car matching :car_id and :id
And a reply with the updated service_record object
DELETE
/v1/cars/:car_id/service_records/:id
This will delete the service_record with matching :car_id and :id and reply with the deleted service_record object
