GET
/v1/cars/:car_id/diagnostic_faults
This will return diagnostic faults if any associated with the car matching the id :car_id
GET
/v1/cars/:car_id/diagnostic_faults/:id
This will the diagnostic fault matching with the id given
POST
/v1/cars/:car_id/diagnostic_faults
This create a new diagnostic fault for the car matching :car_id
And a reply with the new diagnostic fault object
PUT
/v1/cars/:car_id/diagnostic_faults/:id
This will update the diagnostic fault matching :car_id and :id
And a reply with the diagnostic fault object
