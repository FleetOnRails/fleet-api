GET
/v1/cars/:car_id/reminders
This will return reminders if any associated with the car matching :car_id
GET
/v1/cars/:car_id/reminders/:id
This will the reminders matching with the id given
POST
/v1/cars/:car_id/reminders
This create a new reminder for the car matching :car_id
And a reply with the new reminder object
PUT
/v1/cars/:car_id/reminders/:id
This update the reminder for the car matching :car_id and :id
And a reply with the updated reminder object
DELETE
/v1/cars/:car_id/reminders/:id
This will delete the reminder with matching :car_id and :id and reply with the deleted expense object
