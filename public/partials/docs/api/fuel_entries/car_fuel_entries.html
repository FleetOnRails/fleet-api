GET
/v1/cars/:car_id/fuel_entries
This will return fuel_entries if any associated with the car matching :car_id
GET
/v1/cars/:car_id/fuel_entries/:id
This will the fuel_entrie matching with the id given
POST
/v1/cars/:car_id/fuel_entries
This create a new fuel_entry for the car matching :car_id
And a reply with the new fuel_entry object
PUT
/v1/cars/:car_id/fuel_entries/:id
This update the fuel_entries for the car matching :car_id and :id
And a reply with the updated fuel_entry object
DELETE
/v1/cars/:car_id/fuel_entries/:id
This will delete the fuel_entry with matching :car_id and :id and reply with the deleted fuel_entry object
