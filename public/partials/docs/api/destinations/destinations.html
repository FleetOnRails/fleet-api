GET
/v1/groups/:group_id/destinations
This will return destinations owned by the group matching :group_id
GET
/v1/groups/:group_id/destinations/:id
This will return destination owned by the group matching :group_id and :id
POST
/v1/groups/:group_id/destinations
This creates a new destination for the group matching :group_id, the api will geocode the address also
And a reply with the new destination object
PUT
/v1/groups/:group_id/destinations/:id
This will update the destination matching :group_id and :id
And a reply with the destination object
DELETE
/v1/groups/:group_id/destinations/:id
This will delete the destination matching :group_id and :id And a reply with the deleted destination object
