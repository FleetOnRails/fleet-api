GET
/v1/service_records/:service_record_id/documents
This will return documents if any associated with the service record matching :service_record_id
GET
/v1/service_records/:service_record_id/documents/:id
This will the document matching with the :service_record_id and :id given
POST
/v1/service_records/:service_record_id/documents
This create a new document for the service_record matching :service_record
And a reply with the new document object
DELETE
/v1/service_records/:service_record_id/documents/:id
This will delete the document with matching :service_record_id and :id and reply with the deleted document object
