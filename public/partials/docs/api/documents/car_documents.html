GET
/v1/cars/:car_id/documents
This will return documents if any associated with the car matching :car_id
GET
/v1/cars/:car_id/documents/:id
This will the document matching with the id given
POST
/v1/cars/:car_id/documents
This create a new document for the car matching :car_id
And a reply with the new document object
DELETE
/v1/cars/:car_id/documents/:id
This will delete the document with matching :car_id and :id and reply with the deleted document object
