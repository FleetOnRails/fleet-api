GET
/v1/cars/:car_id/diagnostic_statistics
This will return diagnostic statistics if any associated with the car matching the id :car_id
GET
/v1/cars/:car_id/diagnostic_statistics/:id
This will the diagnostic statistic matching with the id given
POST
/v1/cars/:car_id/diagnostic_statistics
This create a new diagnostic statistic for the car matching :car_id
And a reply with the new diagnostic statistic object
