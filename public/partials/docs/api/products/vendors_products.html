GET
/v1/vendors/:vendor_id/products
This will return the products matching :vendor_id
GET
/v1/vendors/:vendor_id/products/:id
This will return the products matching :vendor_id and :id
POST
/v1/vendors/:vendor_id/products
This create a new product for the vendor matching :vendor_id Here is the POST data
And a reply with the new product object
PUT
/v1/vendors/:vendor_id/products/:id
This will update the product with the matching :id
And a reply with the updated product
DELETE
/v1/vendors/:vendor_id/products/:id
This will delete the vendor matching :vendor_id product with the matching :id and respond with the deleted product object
